# 🛡️ ИИ-Ассистент по страховому законодательству РК

Интеллектуальный ассистент для внутреннего использования в страховой компании. Консультирует сотрудников по вопросам страхового законодательства Казахстана на основе загруженных .docx документов.

## Возможности

- 📂 Автоматическая загрузка и разбивка законов на статьи
- 🔍 Умный поиск по документам (семантический + по ключевым словам)  
- 💬 Консультации с цитированием статей и указанием источников
- 📥 Загрузка новых законов через интерфейс
- 📚 Встроенный глоссарий страховых терминов

## Как запустить

1. Склонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Получите бесплатный API-ключ в [Google AI Studio](https://aistudio.google.com/apikey)
4. Запустите: `streamlit run app.py`

## Использованные технологии

- **Языковая модель:** Google Gemini 2.5 Flash
- **Эмбеддинги:** Sentence Transformers (paraphrase-multilingual-MiniLM-L12-v2)
- **Разбивка текста:** LangChain RecursiveCharacterTextSplitter
- **Поиск:** FAISS векторное хранилище
- **Интерфейс:** Streamlit

## Загруженные документы

7 законов Республики Казахстан по страхованию (.docx), включая:
- Обязательное страхование туриста
- Обязательное страхование перевозчика перед пассажирами
- ГК РК Глава 40 (перевозка)
- ОГПОВТС (владельцы транспортных средств)
- Обязательное страхование нотариусов
- Обязательное страхование опасных объектов
- Обязательное экологическое страхование

## Ограничения

- Бесплатный API Google имеет суточный лимит запросов (~1500 в день для Gemini 2.5 Flash)
- Для рабочего режима рекомендуется подключить платный тариф (Pay-as-you-go)
